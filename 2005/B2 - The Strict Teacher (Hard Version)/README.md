## Problem Statement

<div class="header"><div class="title">B2. The Strict Teacher (Hard Version)</div><div class="time-limit"><div class="property-title">time limit per test</div>1.5 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p><span class="tex-font-style-bf">This is the hard version of the problem. The only differences between the two versions are the constraints on $$$m$$$ and $$$q$$$. In this version, $$$m, q \le 10^5$$$. You can make hacks only if both versions of the problem are solved.</span></p><p>Narek and Tsovak were busy preparing this round, so they have not managed to do their homework and decided to steal David's homework. Their strict teacher noticed that David has no homework and now wants to punish him. She hires other teachers to help her catch David. And now $$$m$$$ teachers together are chasing him. Luckily, the classroom is big, so David has many places to hide.</p><p>The classroom can be represented as a one-dimensional line with cells from $$$1$$$ to $$$n$$$, inclusive.</p><p>At the start, all $$$m$$$ teachers and David are in <span class="tex-font-style-bf">distinct</span> cells. Then they make moves. During each move </p><ul> <li> David goes to an adjacent cell or stays at the current one. </li><li> Then, each of the $$$m$$$ teachers simultaneously goes to an adjacent cell or stays at the current one. </li></ul><p>This continues until David is caught. David is caught if any of the teachers (possibly more than one) is located in the same cell as David. <span class="tex-font-style-bf">Everyone sees others' moves, so they all act optimally.</span></p><p>Your task is to find how many moves it will take for the teachers to catch David if they all act optimally.</p><p>Acting optimally means the student makes his moves in a way that maximizes the number of moves the teachers need to catch him; and the teachers coordinate with each other to make their moves in a way that minimizes the number of moves they need to catch the student.</p><p>Also, as Narek and Tsovak think this task is easy, they decided to give you $$$q$$$ queries on David's position.</p></div><div class="input-specification"><div class="section-title">Input</div><p>In the first line of the input, you are given a single integer $$$t$$$ ($$$1 \le t \le 10^5$$$) — the number of test cases. the number of cells on the line, the number of teachers, and the number of queries.</p><p>In the second line of each test case, you are given $$$m$$$ distinct integers $$$b_1, b_2, \ldots, b_m$$$ ($$$1 \le b_i \le n$$$) — the cell numbers of the teachers.</p><p>In the third line of each test case, you are given $$$q$$$ integers $$$a_1, a_2, \ldots, a_q$$$ ($$$1 \le a_i \le n$$$) — David's cell number for every query.</p><p>It is guaranteed that for any $$$i$$$, $$$j$$$ such that $$$1 \le i \le m$$$ and $$$1 \le j \le q$$$, $$$b_i \neq a_j$$$.</p><p>It is guaranteed that the sum of values of $$$m$$$ over all test cases does not exceed $$$2 \cdot 10^5$$$.</p><p>It is guaranteed that the sum of values of $$$q$$$ over all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div
